# Adding Network to Wallet

To interact with WireFluid using an EVM wallet such as **MetaMask**, the network must be added manually. This connection enables you to send transactions, deploy contracts, and interact with dApps directly on the WireFluid chain.

---

## Connection Parameters

Enter the following details exactly as shown to configure your wallet.

| Parameter           | Value                        |
| :------------------ | :--------------------------- |
| **Network Name**    | WireFluid Testnet            |
| **RPC URL**         | `https://evm.wirefluid.com`  |
| **Chain ID**        | `92533`                      |
| **Currency Symbol** | WIRE                         |
| **Block Explorer**  | `https://wirefluidscan.com/` |

---

## Steps to Add WireFluid to MetaMask

1.  **Open MetaMask**
    Click the fox icon in your browser toolbar and click the **network selector** (top-left corner).

2.  **Add Network**
    Select **Add network** from the dropdown, then click **Add a network manually** at the bottom of the list.

3.  **Enter Details**
    Copy and paste the **Connection Parameters** from the table above into the corresponding fields.

4.  **Save & Switch**
    Click **Save**. You can now switch to the **WireFluid Testnet** network.

---

## Performance & Gas Notes

Once connected, your wallet will interact with the WireFluid high-performance network.

- **Fast Finality:** Transactions on WireFluid finalize in approximately **5 seconds**.
- **Low Cost:** Average gas fees are extremely low, ranging from **0.00024 to 0.0017 WIRE** per transaction.
- **Compatibility:** Existing Ethereum accounts and private keys work seamlessly on WireFluid without any reconfiguration.
