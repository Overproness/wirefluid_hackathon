# WIRE Token

**WIRE** is the native asset of the WireFluid blockchain. It pays for transactions, secures the network through staking, and powers governance, one token at the center of the chain’s economy.

---

## At a glance

|                |                              |
| -------------- | ---------------------------- |
| **Symbol**     | WIRE                         |
| **Base unit**  | awire (18 decimals)          |
| **Chain role** | Gas, staking, and governance |

## Native by design

WIRE is not a wrapped or bridged asset. It is issued and used natively on WireFluid. All protocol-level rewards, slashing, and fee flows are in WIRE, so the same asset aligns validators, developers, and users. Whether you interact via an EVM wallet or WireFluid-native tools, WIRE is the common denominator.
