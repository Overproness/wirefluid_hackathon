# WireFluid (Ping.pub) Explorer

**WireFluid Ping Explorer** is a specialized instance of the Ping.pub blockchain explorer tailored for the WireFluid ecosystem. It provides a lightweight, user-friendly interface to explore on-chain data for the WireFluid Testnet and Mainnet, including blocks, transactions, contracts, and account activity.

Built on the Ping.pub platform, a multi-chain block explorer and web wallet for Cosmos SDK and EVM-compatible networks, WireFluid Ping Explorer makes it easy to:

- Search for blocks, transactions, and addresses
- Inspect token transfers and smart contract events
- View validator and network statistics

Use WireFluid Ping Explorer alongside **WireScan** to track activity, debug contracts, and monitor network performance as you build on WireFluid.
